  rooms = {

* First user becomes "host"
* Others become "participants"
* Store role in backend
* Send participants list to all users

* Update list in real-time when users join or leave

* Store videoId in state

6. PLAY / PAUSE SYNC

Frontend:

* Host emits "play" and "pause"
  Backend:
* Validate host role
* Broadcast event to room
  Frontend:
* Listen to "play" → player.playVideo()
* Listen to "pause" → player.pauseVideo()

7. SEEK SYNC (IMPORTANT)

* Extract videoId from URL
* Emit "change_video"
